# Отчет по задаче: Настройка слоев карты (Layer Control) и базовых подложек

## Что было сделано:
1. В `App.tsx` добавлены импорты `LayersControl` и `LayerGroup` из `react-leaflet`.
2. Внутри `MapContainer` добавлен компонент `<LayersControl position="topright">`.
3. Добавлены три базовых слоя (`LayersControl.BaseLayer`):
   - **Спутник (Esri)**: установлен активным по умолчанию (атрибут `checked`).
   - **Темная (CartoDB)**.
   - **Схема (OSM)**.
4. Отрисовка русел рек (массив `riverBeds`) обернута в `<LayersControl.Overlay name="Затопленные русла" checked>` с использованием `<LayerGroup>` для группировки элементов.
5. Отрисовка исторических точек (массив `historicalPoints`) обернута в `<LayersControl.Overlay name="Исторические локации" checked>` аналогичным образом.
6. Геолокация, как видно по текущему коду `App.tsx`, не добавлялась явно в этом шаге, но интерфейс слоев выведен в верхний правый угол (`position="topright"`), чтобы избежать конфликтов с другими возможными элементами управления.

## Статус:
Код успешно обновлен. Функционал переключения базовых карт и отключения/включения оверлеев добавлен. Горячая замена модулей (HMR) должна применить изменения на клиенте автоматически.

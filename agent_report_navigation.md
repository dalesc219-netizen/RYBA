# Отчет по задаче: Навигация к цели и сохранение пользовательских точек (Waypoints)

## Что было сделано:

1. **Прокладка маршрута и расчет дистанции**: 
   - В состояние добавлена переменная `targetPoint`.
   - Логика кнопки «Маршрут» в попапах обновлена: теперь она устанавливает выбранную точку в качестве цели и, при необходимости, запрашивает GPS-координаты пользователя.
   - Между позицией пользователя (`userLocation`) и целью рисуется красная пунктирная линия (`<Polyline pathOptions={{ color: 'red', weight: 4, dashArray: '10, 10' }} />`).
   - Добавлена плавающая панель (баннер) в верхней части экрана, отображающая название цели и точное расстояние до нее (в метрах, если меньше 1 км, и в километрах, если больше). На панели также есть кнопка сброса цели (иконка X).

2. **Функция «Сохранить мою точку»**:
   - В левом нижнем углу добавлена крупная желтая кнопка «Сохранить точку» с иконкой `MapPinPlus`.
   - При клике приложение запрашивает геолокацию. В случае успеха появляется `window.prompt` для ввода названия.
   - Если название введено, точка добавляется в массив `customPoints` (React State) и дублируется в `localStorage` по ключу `rybinka_my_points`.
   - При загрузке приложения точки корректно восстанавливаются из `localStorage` с использованием хука `useEffect`.

3. **Отображение личных точек на карте**:
   - Пользовательские точки выведены в отдельный оверлей `<LayersControl.Overlay name="Мои точки" checked>`.
   - Для их отображения используется кастомная иконка золотой звезды (`Star` из `lucide-react`).
   - Внутри попапа личной точки добавлены кнопки «Маршрут» и «Удалить точку». При удалении обновляется и состояние, и `localStorage`, а также сбрасывается цель маршрута, если она указывала на удаленную точку.
   - Реализована базовая отказоустойчивость: если GPS недоступен, выводятся соответствующие уведомления через `alert`.

## Статус:
Функционал навигации (расстояние до точки) и сохранения собственных точек успешно внедрен. Приложение теперь может работать как простейший картплоттер. Изменения применены.
